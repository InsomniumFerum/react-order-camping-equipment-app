## About The Project

Builded by React. Choose product below anything you want. After that on the top right-hand corner with card button you can order products that you choose before. 

[Project-Demo](https://react-order-camping-equipment-app.netlify.app)

![Screenshot from 2021-05-22 19-22-57](https://user-images.githubusercontent.com/75113766/120231666-863e3100-c25a-11eb-9ce8-349633f84030.png)

#### React Used

* Props
* Hooks (useState, useReducer, useRef)
* Forms
* Custom Hook
* Error Handling
* Loading State
* Fetch with async
* POST and GET data via Firebase

### Built With
* [React](https://www.javascript.com/)
* [HTML](https://html.com/)
* [CSS](https://css.com/)
